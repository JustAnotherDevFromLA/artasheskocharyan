About Me

I am a Software Quality Assurance Engineer based out of Los Angeles, CA. I have a B.S. in Computer Science from California State Polytechnic University - Pomona. I have a deep and thorough understanding of modern QA methodologies, which I use regularly to support and improve software quality within my Agile team. My duties as a Software QA Engineer include manual and automated testing for Web and Mobile apps. I'm also responsible for owning the deployment process, acting as the de-facto release manager by controling code flow between various environments. I make sure to spend time researching the best available frameworks and tools on a consistent basis. I have extensive experience writing various automation scripts using Java, JavaScrpit and Python. Selenium is my go to framework for fast and reliable implementation of test cases. I use Testrail for organizing and managing various Test Suites used for functional and regression testing. I love to learn and explore new technologies and aim to constantly expand my depth of knowledge of relavant tools and frameworks in order to optimize the SDLC for my team.
